C++ AMP Demo for GPU Computing

Requires Visual Studio 2015 to build
Requires Windows SDK (Windows 10 SDK is included in VS2015)
Requires DirectX 11 graphic card with at least limited (WDDM 1.1) double precision support
Optional Kinect for Windows SDK 1.7

For Optional Kinect Control:
Requires Kinect for Windows sensor

GUI is based on Microsoft Hilo project
